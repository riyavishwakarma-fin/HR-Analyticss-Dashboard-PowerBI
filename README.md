# HR-Analyticss-Dashboard-PowerBI
PowerBI dashboard analyzing employee attrition with insights across age, salary, job roles, and deparments.

#📌Project Overview
This project focuses on analyzing employee attrition using Power BI. The goal is to uncover patterns and insights that can help organizations understand why employees leave and support better decision-making.

#🎯Objectives
- Analyze overall employee attrition
- Identify key factors affecting attrition (age, salary, job role, education)
- Provide interactive insights for HR decision-making
 
#📊Dashboard Features
- Total Employee Count, Attrition & Attrition Rate
- Attrition analysis by:
  - Age Group
  - Salary Slab
  - Education
  - Job Role
- Department-wise filtering
- Interactive visuals for deeper exploration
  
#🔍Key Insights
- Higher attrition observed in lower salary bands
- Majority of exits fall within the 26–35 age group
- Certain job roles show significantly higher attrition
- Department-level variations highlight areas of concern
 
#Tools & Technologies
- Power BI
- Data Visualization
- Data Cleaning & Transformation
